Péter-Robison-Ackermann:

PRA m n:
	m = 0 ⇒ n + 1
	m > 0 ⇒
		n = 0 ⇒ PRA (m - 1) 1
		n > 0 ⇒ PRA (m - 1) (PRA m (n - 1))

Implementation:

PRA m n ↦ r
	pm ≔ sub m 1
	pn ≔ sub n 1
	nn ≔ add n 1
	PRA_m_pn ≔ PRA m pn
	PRA_pm ≔ PRA pm 1
	PRA_mn ≔ PRA pm PRA_m_pn
	f_PRA_pm ↦ PRA_pm
	f_PRA_mn ↦ PRA_mn
	if_n ≔ if n f_PRA_mn f_PRA_pm
	f_nn ↦ nn
	if_m ≔ if m if_n f_nn
	r ≔ if_m

Mutual recursion:

even? en ↦ even?en
	true ↦ 1
	ask_odd ↦ odd?pen
		pen ≔ sub en 1
		odd?pen ≔ odd? pen
	even?enf ≔ if en ask_odd true
	even?en ≔ even?enf

odd? on ↦ odd?on
	false ↦ 0
	ask_even ↦ even?pon
		pon ≔ sub on 1
		even?pon ≔ even? pon
	odd?onf ≔ if on ask_even false
	odd?on ≔ odd?onf

Eulicd's algorithm for Extended Greatest Common Divisor:

Euclid a b ↦ gcd x y
	d r ≔ div a b
	base ↦ b 0 1
	recurse ↦ gcd' y' x"
		gcd' x' y' ≔ Euclid b r
		y" ≔ mul y' d
		x" ≔ sub x' y"
	next ≔ if r recurse base
	gcd x y ≔ next
